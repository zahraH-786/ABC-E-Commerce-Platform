ABC E-Commerce Platform

This project is a web-based e-commerce application developed using ASP.NET Core MVC and Entity Framework Core. It provides an online shopping platform where customers can browse products, manage their shopping cart, place orders, and securely register or log in using ASP.NET Identity. The application also includes administrative functionality for managing products, categories, inventory, orders, and user accounts.
Features
• User registration and login with ASP.NET Identity
• Role-based authentication and authorization
• Product catalogue with categories
• Shopping cart functionality
• Order placement and management
• Product and inventory management
• Customer account management
• Administrator dashboard
• Responsive user interface
• SQL Server database integration using Entity Framework Core

Technologies Used
• ASP.NET Core MVC
• C#
• Entity Framework Core
• SQL Server (LocalDB)
• ASP.NET Identity
• HTML
• CSS
• Bootstrap
• JavaScript

Getting Started
1. Clone the repository.
2. Open the solution in Visual Studio 2022.
3. Update the connection string in appsettings.json if required.
4. Open the Package Manager Console and run:

Update-Database
5. Build and run the application.

Project Structure
• Controllers – Handles application logic and user requests.
• Models – Represents the application's data models.
• Views – User interface built using Razor Views.
• Data – Database context and Entity Framework configuration.
• Areas – Identity pages for authentication and authorization.
• wwwroot – Static files including CSS, JavaScript, and images.
